VIM-CMake-Project v1.1.5
===============

About
=====
CMake project management plugin for VIM editor.

[![VCP](http://i.imgur.com/wGeVbl.png)](http://i.imgur.com/wGeVbl.png)

Installing
==========
  
    $ make install

Specify the DESTDIR variable to install it to the other directory.
